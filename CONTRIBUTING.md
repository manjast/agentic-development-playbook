# Contributing

Thanks for your interest. This repo is intentionally lean; changes should
reduce drift and improve clarity without adding heavy process.

## How to contribute
- Open an issue for non-trivial changes.
- Keep edits minimal and provider-agnostic.
- If you add or change a rule, update the README and relevant templates.
- If you add a template field, update the examples.
- If you add a tool name to the README "Works with" section, include a link to the tool's docs that confirm the instruction filename + behavior, and note the verification date.
